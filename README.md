# AI Finance Assistant

An AI-powered personal finance analysis project.

## Project Overview

Many people feel that they do not spend much money, yet their savings fail to grow as expected.

This project explores how AI can help individuals understand spending habits, identify unnecessary expenses, and make better financial decisions.

## Objectives

* Track daily expenses
* Categorize spending patterns
* Generate AI-assisted financial insights
* Improve budgeting decisions
* Build a personal finance knowledge base

## Project Structure

* data/ — raw expense records
* prompts/ — prompts used for AI analysis
* reports/ — weekly and monthly reports
* screenshots/ — charts and visualizations

## Tools

* ChatGPT
* Excel
* GitHub
