# AI Expense Tracker

## Project Background

I often feel that I don't spend much money, but my savings do not increase as expected.

This project aims to track daily expenses, analyze spending patterns, and explore how AI can help with personal financial management.

## Goals

- Record daily expenses
- Categorize spending
- Analyze monthly spending habits
- Generate AI-powered insights
- Improve personal budgeting decisions

## Tools

- Excel
- ChatGPT
- GitHub
